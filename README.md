# 🚀 Typing Speed Test App

A modern **Typing Speed Test Web Application** built using **Flask (Python)** and **HTML, CSS, JavaScript**.
This app helps users measure their typing speed, accuracy, and performance in real time with an interactive interface.

---

## 📌 Description

This project is inspired by popular typing platforms like Monkeytype. It allows users to practice typing using random paragraphs and instantly view their performance metrics such as **WPM (Words Per Minute)**, **accuracy**, and **correct words count**.


---

## 🎯 Features

* ⏱️ **Real-time Timer** (30 seconds)
* ⚡ **Live WPM Calculation**
* 🎯 **Accuracy Tracking**
* ✅ **Correct Words Count**
* 🧠 **Random Paragraph Generator**
* ✍️ **Character Highlighting (correct/wrong)**
* ⛔ **Auto-stop when typing completes**
* 🎉 **Instant Result Display**
* 🔄 **Restart Test Option**

## 📊 How It Works

1. User starts typing → timer starts automatically
2. App tracks:

   * Words per minute (WPM)
   * Accuracy percentage
   * Correct words
3. Test ends when:

   * Time runs out ⏱️
   * OR user completes the paragraph 🎯
4. Results are displayed instantly 🎉
